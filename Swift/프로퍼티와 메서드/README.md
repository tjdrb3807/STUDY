# 프로퍼티와 메서드
`프로퍼티`는 클래스, 구조체 또는 열거형 등에 관련된 값을 뜻한다. 

`메서드`는 특정 타입에 관련된 함수를 뜻한다. 

변수나 상수, 함수 등이 어떤 목적으로 쓰이느냐, 어디에서 어떻게 쓰이느냐에 따라 용어가 조금씩 달라질 뿐이다.
